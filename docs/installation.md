# Installation

## Stable release

To install FairSharer, run this command in your terminal:

```sh
uv add fairsharer
```

Or if you prefer to use `pip`:

```sh
pip install fairsharer
```

## From source

The source files for FairSharer can be downloaded from the [Github repo](https://github.com/Hebivi/fairsharer).

You can either clone the public repository:

```sh
git clone git://github.com/Hebivi/fairsharer
```

Or download the [tarball](https://github.com/Hebivi/fairsharer/tarball/master):

```sh
curl -OJL https://github.com/Hebivi/fairsharer/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd fairsharer
uv pip install .
```
